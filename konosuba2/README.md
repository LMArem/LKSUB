![poster](poster.jpg)

## STORY

热爱游戏的家里蹲少年·佐藤和真的人生，因交通事故（!?）而轻易闭幕……本该是这样，但当他醒来之时，眼前有一位自称是女神的美少女。

"喂，我有点好事 要告诉你。要去异世界吗？只带一样你喜欢的东西没问题喔。""那，我就带着你好了。"

由此开始，在异世界转生的和真的魔王讨伐大冒险开始了……虽然是这么 想的，但他却为了获得衣食住行而开始劳动！想要平稳度日的和真，却由于女神引起的各种问题，而终于被魔王军盯上了!?
